### 2.2. 핵심 추가 내용

**신규 섹션: v3 변경 배경 (§2)**
- 6개 BLOCKING Gap 정의 (B1~B6)
- 각 Gap별 v2 상태 vs v3 해결 방법 비교표
- 변경 파일 목록 및 변경 유형

**신규 섹션: 자율 실행 프로세스 (§3)**
- v2 vs v3 실행 흐름 비교 (사람 개입 5회 → 0~1회)
- 7단계 상세 + 각 단계별 사람 개입 여부
- 핵심 알고리즘 3개 요약 (탐색/산출/에러수정)

**신규 섹션: v3 검증 방법 (§7)**
- 시뮬레이션 검증 기준
- 실제 테스트 검증 KPI (자율 실행 완료율 90% 등)

**측정 기준 추가 (§4.4)**
- 정량: "자율 실행 성공률 90% 이상"
- 정성: "클래스명 입력만으로 테스트 생성 가능"

**워크플로우 변경 (§4.5)**
- v2: 8단계 (Claude → Skills 참조 5회가 분리)
- v3: 11단계 (자율 실행 내부 단계 명시, 자동 수정 포함)

### 2.3. 삭제/변경 없는 내용

- 배경 및 문제점 (§4.1)
- 추진 필요성 (§4.2)
- 적용 목표 1~3단계 (§4.3)
- 기술 스택 (§5)
- 추진일정 골격 (§6)

## 6. 자율 실행 수준 비교

### 6.1. 사람 개입 지점 비교
