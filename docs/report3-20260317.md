## High (10건) — 코드 예시 오류/정합성

### H1. `references/examples/service-test-example.md` — PasswordEncoder 의존성 불일치

예시 소스 클래스에 `PasswordEncoder` 의존성이 있지만, 실제 `UserService`에는 없음 (`AuthService`가 담당).
에이전트가 이 예시를 참조하면 존재하지 않는 `PasswordEncoder` Mock을 생성할 수 있음.

**수정 방안**: 예시를 실제 UserService 구조(UserMapper만 의존)에 맞추거나, "가상의 예시"임을 명시

### H2. `references/examples/controller-test-example.md` — URL 경로 불일치

예시 컨트롤러 URL:
```
@GetMapping("/list"), @GetMapping("/detail/{id}"), @PostMapping("/create")
@PostMapping("/update/{id}"), @PostMapping("/delete/{id}")
```

실제 UserController URL (REST 표준):
```
@GetMapping, @GetMapping("/{id}"), @PostMapping
@PutMapping("/{id}"), @DeleteMapping("/{id}")
```

**수정 방안**: 예시를 REST 표준 패턴으로 변경

### H3. `references/examples/controller-test-example.md` — User.builder() 사용

`User.builder().id(id).name(name)...build()` 패턴 사용. 실제 User 클래스에는 `@Builder` 없음 (setter 방식).

**수정 방안**: setter 기반 객체 생성으로 변경

### H4. `references/examples/service-test-example.md` — deleteById Mock 패턴 오류

```java
// 현재 (오류): void 메서드용 패턴
doNothing().when(userMapper).deleteById(TEST_USER_ID);

// 수정: int 반환 메서드에 맞는 패턴
when(userMapper.deleteById(TEST_USER_ID)).thenReturn(1);
```

### H5. `verification/test-execution.md` — anyString() 잘못된 사용 (5개소)

`anyString()`을 실제 메서드 호출 인자로 사용. Mockito matcher는 `when()`/`verify()` 안에서만 사용 가능.

```java
// 현재 (오류): L115, L119, L135, L136, L139, L163
userService.getUserById(anyString());

// 수정
userService.getUserById("testId");
```

### H6. `verification/compile-check.md` L99 — 닫는 괄호 누락

```java
// 현재 (오류)
@ExtendWith(MockitoExtension.class

// 수정
@ExtendWith(MockitoExtension.class)
```

### H7. `references/examples/controller-test-example.md` — UserService import 누락

`@MockBean private UserService userService;` 선언은 있지만, import 목록에 `UserService` 없음.

### H8. `references/examples/util-test-example.md` — Constructor import 누락

`Constructor<?>` 사용하지만 `import java.lang.reflect.Constructor;` 누락.

### H9. `CLAUDE.md` — 작업 이력 미반영

작업 이력이 2026-03-09까지만 기록됨. 2026-03-17 작업 미반영:

- 테스트 수 산출 공식 변경

### H10. `CLAUDE.md` — "다음 작업 예정" 섹션 오래됨
